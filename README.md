
Entre as principais funcionalidades implementadas, destacam-se:
✅ Layout 100% Responsivo
 O site se adapta perfeitamente a diferentes tamanhos de tela (desktop, tablet e mobile), garantindo boa usabilidade em qualquer dispositivo.
✅ Menu de Navegação Interativo
 Menu estruturado para facilitar a navegação entre as seções do site, proporcionando uma experiência fluida e intuitiva.
✅ Sistema de Formulário Funcional
 Implementação de formulário com validação em JavaScript, garantindo que os dados inseridos pelo usuário sejam preenchidos corretamente antes do envio.
✅ Abertura de Chamados/Orçamentos
 Estrutura preparada para que o usuário possa solicitar serviços de forma organizada e prática.
✅ Efeitos Visuais e Animações em CSS
 Aplicação de efeitos hover, transições suaves e animações para tornar a interface mais moderna e dinâmica.
✅ Scroll Suave entre Seções
 Navegação fluida ao clicar nos links do menu, melhorando a experiência do usuário.
✅ Organização Modular do Código
 Separação clara entre HTML, CSS e JavaScript, priorizando legibilidade, manutenção e boas práticas de desenvolvimento.
